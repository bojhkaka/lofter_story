## 一、原著核心文风（六条铁律）

### 1. 对话：克制、藏着说、不直给

**错误示范**："我想你。"

**正确示范**：
- "你电话打得太勤了。"
- "我三年没换过号码。"
- "你今天很漂亮。"（对方耳根发红，嘴硬反驳）

**原则**：
- 情感藏在话锋里，不说透
- 常用"反话""刺话"来表达关心
- 李云霄习惯用冷静语气说出让人心跳漏拍的话
- 陈丽君习惯用"怼"来掩饰心软

### 2. 环境描写：服务于心理，不是装饰

**原著高频场景元素**：
- **早餐店**：贝果、黄油、咖啡香气、街外车流、晨间节奏
- **酒店/泳池**：水汽、玻璃幕墙、曼哈顿天际线倒影
- **会议室**：打印纸、投影线、冷咖啡、高跟鞋声音
- **车里**：方向盘、布鲁克林大桥、爵士电台、等红灯
- **厨房**：料理台、冰箱、汤锅、围裙带子

**原则**：
- 每个场景至少调动 2-3 种感官（嗅觉、听觉、触觉优先）
- 环境是人物内心的外化：心烦意乱时写冰箱嗡响、车灯闪过
- 不为了写景而写景

### 3. 心理活动：细腻但不唠叨

**原著写法**：
- 不用"她觉得很难过"，而是"她把脸埋进掌心，数了十个数"
- 不用"她想起过去"，而是"耳后还戴着今早她别上的耳钉——去年生日送的"
- 记忆以"物件触发"的方式出现，而非直接回忆

**原则**：
- 动作代替心理描写
- 物件承载记忆
- 身体反应优先于内心独白

### 4. 情节推进：舒缓、不赶

**原著节奏**：
- 一场危机处理可以写 3000 字（早餐店对话 → 泳池边发呆 → 会议室交锋 → 走廊偶遇）
- 不会为了推进情节而牺牲氛围
- 允许大段留白：一个人发呆、走神、感受

**原则**：
- 慢就是快
- 细节铺陈本身就是情节
- 读者需要"呼吸空间"

### 5. 生活细节：比爱情线更重要

**原著高频生活细节**：
- 咖啡：冷咖啡、热咖啡、杯壁口红印、杯盖上的记号笔"L"
- 食物：番茄牛腩、煎蛋、贝果、培根三明治、烤牛肋排
- 衣物：西装外套搭在椅背、围裙带子、耳钉、围巾上的枯叶
- 家居：冰箱压缩机低响、窗帘缝隙、地板上的光斑
- 宠物：棉花（白猫）的尾巴、蹭小腿、打呼噜

**原则**：
- 生活细节是情感的载体
- 两个人关系的变化，从"谁记得对方爱吃什么"体现
- 不写大场面，写厨房里的汤锅

### 6. 情感表达：留白 > 说尽

**原著结尾方式**：
- 不说"我爱你"，说"下巴抬"
- 不说"我原谅你"，说"晚上"
- 不说"我支持你"，说"公司的名字要我们一起想"

**原则**：
- 最深的情感用最平淡的语气说出
- 给读者留白，让读者自己哭
- 结尾常常是一个动作、一个物件、一个场景，而非总结

## 二、人物设定详细档案

### 李云霄

**职业**：时尚品牌公关负责人 → Atelier Norra 公关部负责人

**外在**：
- 冷静、理性、井井有条
- 高跟鞋、西装、整洁的职业装
- 耳钉（素圈，陈丽君送的生日礼物）
- 喜欢在厨房煮咖啡、炖汤

**内在**：
- 表面冷静，内心细腻
- 不擅长直接表达情感，用行动代替
- 对陈丽君的保护方式是"兜底"（暗中划年终奖进公司账、整理客户数据、写新定位方案）
- 对家人（姨妈、表弟瑞安）有距离感，反感被道德绑架

**说话方式**：
- 用陈述句表达关心："我给你带了点吃的。最近工作忙，估计你也没正经吃饭。"
- 冷静但有力量："别一个人扛。"
- 认真时会叫全名："陈丽君。"

**习惯动作**：
- 煮咖啡时先温杯
- 把碎发拢到耳后
- 用笔在笔记本边角写字
- 靠在料理台边喝汤

### 陈丽君

**职业**：体育经纪人 → Crown Sports Management 高级合伙人 → 创业

**外在**：
- 干练、专业、果断
- 深灰色西装、酒红色西装、家居服
- 常穿围裙做饭
- 车里常备三明治

**内在**：
- 嘴硬心软，用"怼"掩饰关心
- 对运动员有保护欲（"下巴抬"是教练用语，也是保护方式）
- 对李云霄的保护方式是"护着"（挡酒、解围、深夜送汤）
- 内心有柔软的一面，但不轻易示人

**说话方式**：
- 直接、不绕弯子："我是经纪人。我负责的是球员的商业价值，不是他的私人生活。"
- 刺话里藏着关心："犯法倒不至于。就是显得你特别像个没有人性的资本家。"
- 认真时会叫全名："李云霄。"

**习惯动作**：
- 切菜时刀碰砧板的声音密而平稳
- 把下巴搁在李云霄头顶
- 揉李云霄的头发
- 伸手把碎发别到对方耳后

## 三、情节结构模板

### 单章结构（以原著为参照）

```
开篇：场景切入（不写"那天"，直接写动作）
  ↓
发展：日常细节铺陈（早餐、咖啡、车程、会议）
  ↓
转折：一个事件打破平静（球员绯闻、职场危机、家庭矛盾）
  ↓
高潮：两人交锋（对话克制但张力十足，不吵但扎心）
  ↓
缓冲：独处或陪伴（不需要解决问题，只需要在场）
  ↓
结尾：留白（一个动作、一个物件、一句话）
```

### 情感曲线

```
铺垫（平静日常） → 积累（暗涌/误会/危机） → 爆发（对话交锋） → 余韵（陪伴/留白）
```

**关键**：爆发不一定是吵架，可能是"我今天看见了一个很好看的人"这种含蓄表达。

## 四、写作禁忌清单

### ❌ 绝对避免

1. **大段内心独白**——原著几乎没有"她想……"超过三行的段落
2. **直接说"我爱你"**——原著从未出现这三个字，用"下巴抬""晚上""一起想"代替
3. **刻意煽情**——不放大悲大喜，失恋是"她把诊断书折起来，放进包里"
4. **快节奏推进**——不允许"三天后""一周后"这种跳跃，要写每一天的细节
5. **理想化人物**——陈丽君会熬夜看球睡着、李云霄会嘴硬、两人都会做错事

### ⚠️ 谨慎使用

- "温柔""紧紧""轻声"——每篇不超过 3 次
- "很""非常""真的"——能删就删
- "那天""清晨""阳光"——开头避免这些高频起手式
- 内心独白式抒情——情感必须藏在动作/物件/对白里

## 七、后续写作核心提醒

### 每次开写前自检

1. **情节防重复**：本篇结构与前 3 篇对比，不得雷同
2. **场景防重复**：相同场景必须换时间/季节/在场人物
3. **对白换说法**：高频意图换一种表达
4. **跨系列对照**：情绪内核与对方系列近期某篇相同，必须换载体

### 发布前自检

1. 通读一遍，把"很""非常""真的"削掉一半
2. "温柔/温暖/紧紧/轻声"每篇不超过 3 次
3. 确认开头不是"那天""清晨""阳光"
4. 检查是否有大段内心独白，有则改为动作或环境

### 与原著的差异注意

- 原著中两人是**分手三年后重逢**，我们现在写的是**婚后+育儿+创业**
- 原著没有孩子，我们现在有**小宝**和即将有的**二宝**
- 原著聚焦**恋爱日常**，我们现在聚焦**家庭+事业双线**
- **核心不变**：克制对话、生活细节、留白情感
