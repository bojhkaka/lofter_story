## 一、系列基调：田园牧歌式理想世界

《一茶一戏》是一个**没有社会压力的世界**——同性之爱如同呼吸般自然，家庭全然接纳，事业虽有挑战但终会成功。

| 维度 | 《一茶一戏》 | 《纽约故事》 |
|---|---|---|
| 世界观 | 架空乌托邦，"没有同性恋的说法" | 现实向，有社会压力和身份焦虑 |
| 冲突烈度 | 低——李云霄"作"→陈丽君哄→甜蜜 | 高——有真实的争吵、冷战、深层矛盾 |
| 情感基调 | 温暖甜蜜为主，冲突轻微 | 甜虐交织，有棱角的都市写实 |
| 对话风格 | 更口语化、更俏皮、更有烟火气 | 更成熟、更克制、更理性 |
| 节奏 | 慢如品茶，"温润"为核心 | 明快但沉稳，"真实"为核心 |
| 结尾方式 | 仪式感金句收束："一茶一戏，一辈子" | 留白场景收束，不说尽 |

## 二、对话风格（一茶一戏独有）

### 核心：对话占比 60%+，短句交锋，撒娇-哄劝模式

**错误示范**（太纽约式）：
> "你今天怎么了？"李云霄声音很轻。
> "没什么。"陈丽君看着窗外。

**正确示范**（一茶一戏式）：
> "陈丽君！"
> "怎么？"
> "你凶我。"
> "我哪有。"
> "你就有。"
> 陈丽君失笑，伸手把她揽过来："好好好，我错了。"

### 撒娇-哄劝固定模式

这是《一茶一戏》最标志性的互动模式，几乎每篇都出现：

```
李云霄撒娇/作 → 陈丽君哄（带点无奈但不油腻）→ 甜蜜收束
```

示例：
- "我不活了，今天《梁祝》你找别人搭戏吧……" → "好好好，我错了。"
- "你又要去义父家蹭饭……" → "一起啊。"
- "你这个人，怎么这样啊……" → "怎样？" → "太好了，好得让人想欺负你。"

**注意**：偶尔可以让陈丽君也撒一次娇，避免模式过于单一。

### 对话中植入信息

避免生硬旁白，在对话中自然交代文化知识和背景：

```
"云霄喜欢喝狮峰山的，说那股兰花香像我家后山。"
（一句话交代：茶叶品种+产地+人物口味）
```

```
"你胃不好，只能喝温性的红茶。我这次让我爸特意留了些九曲红梅，适合你。"
（同时展现：关怀+茶知识+家庭关系）
```

## 三、人物设定（一茶一戏独有）

### 陈丽君

**职业**：小百花越剧团 小生演员 → 梅花奖得主

**性格画像**：阳光温暖型"完美恋人"，兼具少年气和成熟感。

**说话方式**：
- 轻松幽默带痞气："李大小姐金枝玉叶，是我这个卖茶叶的粗人不会说话。"
- 表达感情时直接真诚："因为我喜欢你啊。"（从十六岁到现在，这句话从未变过）
- 哄人有套路感但不油腻："好好好，我错了" → 行动补偿（泡茶、带礼物）

**行为标签**：
- 五点起来熬红糖姜茶
- 化妆间永远飘着茶香
- 背后环腰抱
- 泡茶/采茶/教采茶

**弱点/可爱之处**：偶尔忘事（忘了李云霄不吃香菜）、被"作"到无奈但甘之如饴

### 李云霄

**职业**：小百花越剧团 花旦演员 → 梅花奖得主

**性格画像**：外表文静、内里倔强的"小作精"，撒娇功力一流。

**说话方式**：
- 撒娇式抱怨为主："你凶我""我不活了"
- 害羞时声音变小、耳朵变红
- 带台州口音的"软软糯糯的尾音"
- 表达深情时反而简洁："一年四季，都一起过。""我要！"

**行为标签**：
- 半夜偷吃冰淇淋
- 胃疼（体寒，反复出现）
- 把脸埋进陈丽君肩窝/颈窝
- 跺脚

**成长线**：从"总觉得我不行"到获得白玉兰奖/梅花奖后的自信

### 配角档案

| 人物 | 身份 | 说话风格 | 经典台词 |
|---|---|---|---|
| 蔡铭（义父） | 越剧老生/闺蜜 | 大嗓门、夸张、爱吐槽 | "义子！义父为你骄傲！" |
| 方圆（小圆子） | 音响师/死党 | 惜字如金、冷面热心 | "你那不叫骄傲，叫蹭热度。" |
| 陈莲君（姐姐） | 茶山管理者/长姐 | 亲切自然、姐姐式调侃 | "谁是你姐？我现在是云霄的靠山。" |
| 外婆 | 越剧迷老太太 | 行家点评+揭短 | "也不知道是谁，小时候天天念叨'君君说'……" |
| 陈父陈母 | 茶农/朴实本分 | 温暖质朴、话不多但每句都实在 | （用行动表达） |

## 四、茶文化元素库

茶是系列的灵魂，每篇至少 3-5 个茶文化细节。

### 茶叶品种
- **龙井**（明前龙井）——核心品种，出现频率最高
- **狮峰山龙井**——李云霄最爱的品种
- **碧螺春**
- **安吉白茶**
- **九曲红梅**——温性红茶，适合胃寒的人

### 茶事活动
- **采茶**：一芽一叶的手法、清明前上山采茶
- **泡茶**：盖碗泡法、温度控制、第一泡洗茶
- **品茶**：闻香、观色、品韵
- **做桂花龙井**：秋天收集桂花，与龙井窨制

### 茶器具
- **盖碗**：最常用的泡茶器具
- **保温杯**：日常随身
- **老井水**：嵊州茶山的水泡茶最好

### 茶与生活
- **茶叶蛋**：陈丽君煮的
- **茶汤色泽**：嫩绿透亮
- **茶香**：兰花香（狮峰龙井）、桂花香（桂花龙井）
- **茶树清香**：茶山空气里的味道

### 茶的功能（叙事层面）
- **情感载体**：泡茶 = 表达爱意，教采茶 = 亲密互动
- **人物标签**：陈丽君 = "隐性茶庄老板"
- **家族纽带**：陈家茶山是家族根基
- **时间标记**：明前茶 = 春天 = 新开始

## 五、越剧元素库

### 经典剧目
- **《梁山伯与祝英台》**（《梁祝》）——核心剧目，出现最多
- **《十八相送》**——《梁祝》选段，十六岁时第一次搭戏
- **《楼台会》**——《梁祝》选段
- **《碧玉簪》**——首演成功后获白玉兰提名
- **《何文秀》**
- **《红楼梦》**——白玉兰奖后复排（贾宝玉+林黛玉）

### 行当术语
- **小生**（陈丽君，反串男角）
- **花旦**（李云霄）
- **老生**（蔡铭）

### 表演元素
- **水袖**：长水袖练习、舞台甩袖
- **练功**：每天清晨练功、压腿、走台步
- **头面**：花旦头饰、化妆过程
- **戏服**：白色礼服上绣茶花（白玉兰奖时）
- **化妆间**：陈丽君的化妆间永远飘着茶香

### 越剧的功能（叙事层面）
- **事业主线**：练功→比赛→获奖→巡演→申报梅花奖
- **情感隐喻**：台上演梁祝，台下是"梁祝"——戏与人生互文
- **文化根基**：越剧+杭州+茶，三者共同构建"江南文化"底色

## 八、情感表达特征（一茶一戏独有）

### "笑着笑着就哭了"

系列最标志性的情感签名，几乎每篇都出现：

- "李云霄终于笑了，眼泪却掉下来"
- "笑着笑着眼泪却掉下来"
- "陈丽君笑了，眼泪也掉了下来"

### 肢体语言代替心理描写

| 动作 | 情感含义 |
|---|---|
| 把脸埋进肩窝/颈窝 | 依赖/撒娇 |
| 背后环腰抱 | 亲密/保护 |
| 桌下握手 | 默契/安慰 |
| 从背后握住手教 | 亲密教学 |
| 耳朵红了 | 害羞（高频细节） |
| 顺着后背像给猫顺毛 | 安抚 |

### 问答递进式情感高潮

重要情感场景采用层层递进的问答结构：

```
"你会一直喜欢我吗？"
"会。从十六岁到现在，十几年了，我一天比一天更喜欢你。"
"那我要是变得更作呢？"
"惯着。"
"我要是老了不漂亮了呢？"
"你八十岁我也给你泡龙井。"
```

## 九、篇章结构模板

### 模式A：日常散文式

```
开场环境（一句话定调）→ 日常互动（撒娇-哄）→ 回忆/闪回 →
家庭场景（饭桌/茶山）→ 深夜谈心 → 环境收束 →
金句结尾："一茶一戏，一辈子。"
```

### 模式B：事业里程碑式

```
好消息传来 → 准备阶段 → 紧张等待 →
颁奖/结果揭晓（高潮）→ 庆祝（朋友/家人）→
二人世界的温情收尾 → 金句收束
```

### 后续可用：家庭成长式（新增）

```
日常铺垫（孩子/父母/生活）→ 一个小事件/小冲突 →
解决过程中的温情细节 → 家庭场景（饭桌/茶山）→
环境收束 → 金句结尾
```

## 十、写作禁忌与自检

### ❌ 绝对避免

1. **过于纽约式**：不写冷冰冰的职场博弈、不写尖锐的社会压力
2. **打破乌托邦设定**：不写"出柜""歧视""社会不接纳"
3. **金句缺席**：每篇结尾必须有仪式感的收束
4. **茶文化/越剧文化缺失**：每篇至少 3-5 个相关细节
5. **撒娇-哄劝模式完全消失**：可以变化但不能没有

### ⚠️ 谨慎使用

- "笑着笑着就哭了"——每篇最多 1-2 次，避免审美疲劳
- 撒娇模式——偶尔让陈丽君也撒一次娇
- 配角戏份——不能喧宾夺主，核心仍是两人互动

### 每篇开写前自检

1. 对话占比是否 ≥ 50%？
2. 茶文化/越剧文化细节是否 ≥ 3 个？
3. 有没有撒娇-哄劝的互动？
4. 结尾有没有仪式感金句？
5. 情感表达是否以外化为主（动作/对话），而非内心独白？
6. 环境描写是否以自然/茶/江南为主？

### 发布前自检

1. "很""非常""真的"削掉一半
2. "温柔/温暖/紧紧/轻声"每篇不超过 3 次
3. 确认开头不是"那天""清晨""阳光"
4. 检查有没有"纽约式"的冷调对话混入
